# NHSBSA Data Hub
 
The Data Hub is an NHSBSA service/prototype that helps users find and understand NHSBSA data products and related information.
 
This repository contains the Alpha prototype built using the NHS Prototype Kit.
 
## Status
 
This is an Alpha prototype and is not production code.
 
## Running locally
 
### Prerequisites
 
- Node.js 22.x or 24.x
- npm
 
### Install dependencies
 
npm install
 
### Start the prototype
 
npm start
 
The prototype runs locally at:
 
http://localhost:3000
 
## Prototype versions
 
Current prototype versions are available under:
 
- /data-hub/mvp/v5
 
Older versions are kept to show design iteration and research history.
